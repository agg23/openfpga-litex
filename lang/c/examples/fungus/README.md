# C Example: fungus

`make`

Fluidly expanding colors and grumbling sound. Shows off an agent-based organic growth mechanism for drawing to the framebuffer.

![](./screenshot.png)

## Controls

| Button | Functionality                                          |
| ------ | ------------------------------------------------------ |
| D-Pad  | Move agents in that direction                          |
| A      | Increases the rate at which the color changes          |
| B      | Select between 3 speeds                                |
| X      | Toggle super grow mode, causing agents to grow outward |
| Y      | Toggle whether only winners will draw to the screen    |
| Select | Pause                                                  |
| Start  | System reset                                           |
